# Feature Requests

## Обзор
Этот документ содержит список идей и предложений по развитию проекта, которые не вошли в основной бэклог. Его цель — структурировать и сохранить потенциальные улучшения, чтобы команда могла вернуться к ним при планировании будущих релизов. Используйте этот документ для обсуждения и оценки новых функций, а также для вдохновения при разработке.

## Фичи для пользователей
- Расширенные фильтры (по языку, длине, типу контента)
- Гибкая система тегов для сюжетов и постов
- Настраиваемые шаблоны промптов
- История переписанных версий текста
- Система комментариев к сюжетам

## Интеграции и источники
- Поддержка RSS и веб-парсинга новостей
- Интеграция с внешними API для аналитики и тональности
- Подключение к системам контент-планирования (Notion, Trello и др.)
- Импорт текстов из файлов и Google Docs

## AI и автоматизация
- Автоматическая генерация заголовков
- Классификация постов по тематикам
- Рекомендации по улучшению рерайта
- Автопубликация по расписанию на основе активности аудитории

## Публикация и дистрибуция
- Одновременная публикация в несколько каналов
- Предпросмотр поста «как в Telegram» до отправки
- Публикация с кастомными кнопками и реакциями
- Архив опубликованных сюжетов с поиском

## Аналитика и метрики
- CTR и вовлечённость постов
- Сравнение переписанного текста с оригиналом
- Метрики по моделям рерайта и их точности
- Автоматические отчёты и дашборды

## DevOps и инфраструктура
- Поддержка S3 для медиа
- Распределённая обработка задач
- Возможность горизонтального масштабирования воркеров
- Интеграция с системами мониторинга (Prometheus, Grafana)
