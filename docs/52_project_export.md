# Экспорт проектов

## Цель
Экспорт позволяет выгрузить настройки проекта без постов: параметры проекта, шаблоны
промтов, источники и снимки пресетов веб-парсера. Это удобно для резервных копий,
переноса конфигурации и обмена шаблонами между средами.

## Где находится
- Страница настроек проекта: кнопки `JSON` и `YAML`.
- Формат выбирается через параметр `?format=` (`json` по умолчанию, `yaml`).

## Что попадает в экспорт
- `project`: основные настройки проекта (канал публикации, локаль, интервалы, модели).
- `prompt_config`: секции промта проекта.
- `sources`: список источников с параметрами фильтрации, ретеншна и снимком пресета.
- `web_presets`: уникальные пресеты, используемые источниками, с конфигурацией.

## Что не попадает в экспорт
- Посты, сюжеты, публикации и медиа.
- Временные статусы сбора (последний запуск, last_synced_id и т.п.).

## Рекомендованный сценарий
1. Перейти в настройки проекта.
2. Скачать JSON или YAML экспорт.
3. Сохранить файл в защищенном месте (без доступа для посторонних).
4. Для восстановления или переноса использовать соответствующий импорт (если реализован).
